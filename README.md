# Parking Lot

Shell app to simulate parking system

## Installation & Download Dependency

```bash
bin/setup
```

## Run functional spec

```bash
bin/run_functional_tests
```

## Run shell App

```bash
bin/parking_lot
```

## Project Structure

```
src/
    handlers/
        parking.go
    models/
        car.go
        parking.go
    service/
        parking.go
    main.go
go.mod
Makefile
```

## Feature

- create parking slot
- cek status parking slot
- new car come to parking
- car exit
- exit command
